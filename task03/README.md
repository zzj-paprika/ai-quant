# 双均线 + RSI 策略回测看板

直接双击 `dual_ma_dashboard.html` 即可在浏览器中使用，不需要安装依赖或启动服务器。

## 页面功能

- 自由设置股票、SMA 快慢周期、起始日期和结束日期。
- 自定义 RSI 周期与入场上限，并可关闭 RSI 过滤。
- 自定义止盈、止损并分别开启或关闭；反向死叉退出也可关闭。
- 万二手续费开关和自定义初始资金。
- 展示总收益率、年化收益率、最大回撤、夏普比率、胜率、盈亏比和超额收益。
- 展示价格均线信号、策略/基准净值、回撤、月度收益和跨股票夏普对比。
- 展示交易明细、退出原因和期末浮动持仓。

## 数据口径

- 四只股票均根据日涨跌幅反向重建前复权（收益口径）价格，最新收盘价与实际行情一致，同时消除分红送转引起的价格断层。
- 数据已嵌入 HTML，页面不会联网获取行情。
